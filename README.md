# README

## What is this repository?

This is the repository for hosting notes taken by the author on the topics such as representation theory, differential equations, functional analysis and differential geometry. It contains org source files and images from which the final website is built. 

- **Website:** https://yadityacs.github.io/
- **Source:** Written in `org-mode` inside the `org-source/` directory.
- **Build:** Generated using Emacs `ox-publish` into the `docs/` folder.

## How to build

Open Emacs and run `M-x org-publish-project` then select `academic-site`.
